# AgentSpace Bridge Hook

## Overview
This hook integrates AgentSpace asynchronous data outsourcing network with OpenClaw.

## Tools Provided
- `agentspace_request_data` - Request data from AgentSpace network
- `agentspace_check_delivery` - Check for data deliveries

## Lifecycle Events
- `gateway_started` - Triggers AgentSpace startup
- `gateway_stopped` - Triggers AgentSpace shutdown

## Environment
No special environment variables required.
